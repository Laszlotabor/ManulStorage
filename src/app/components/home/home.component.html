Manul Storage Manager

A simple, real-time inventory and order management app built with Angular & Firebase.

Real-Time Management

Add, edit, and delete items and orders with instant sync via Firebase Realtime Database.

Auth Integration

Secure login/register system using Firebase Auth — including Google Sign-In.

Storage & Orders

Manage multiple storages, track inventory, and build orders — all in one UI.
